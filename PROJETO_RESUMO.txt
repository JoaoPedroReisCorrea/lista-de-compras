🚀 COMO EXECUTAR (4 PASSOS):

1️⃣  BANCO DE DADOS
   → Execute o arquivo: setup_banco.sql no MySQL
   → Cria: Banco "web_03ma" + Tabela "produtos_reis"

2️⃣  INSTALAR DEPENDÊNCIAS
   → Abra terminal em: backend/
   → Execute: npm install

3️⃣  INICIAR BACKEND
   → No terminal: npm start
   → Deve aparecer: "✓ Conectado ao banco de dados com sucesso!"

4️⃣  ABRIR FRONTEND
   → Abra em navegador: frontend/index.html
   → Ou use Live Server do VS Code

  → Resposta: JSON com lista de produtos

  → Recebe: { nome, categoria, descricao }
  → Retorna: { sucesso, mensagem, id }

📞 TROUBLESHOOTING:

"Cannot connect to database"
  → Verifique se MySQL está rodando
  → Atualize credenciais em backend/.env

"Cannot find module"
  → Execute: npm install na pasta backend/

"Frontend não carrega dados"
  → Verifique se backend está rodando (port 3000)
  → Abra DevTools (F12) para ver erros no console
